The DOM (Document Object Model) is the structure that the browser uses to make sense of the web document it is displaying. It represents HTML in a logical tree-like structure It is the data representation of all the objects that compise the structure and content of a document on the web It is a programming interface for HTML and XML documents Represents the page so that programs can change the document strucutre, style and content Represents the documend as nodes and objects It is an object-oriented representation of the web document, which can be modified by a scripting language such as JS JS uses the DOM to access the document and its elements Every element in a document (including the dcument itself,head, body, etc) is part of the DOM for that document As an example: will display an alert when the document is loaded The DOM provides the missing link that ties your HTML and CSS with JS It also provides access one level up to the 'window' and 'document' objects - 'window' is the root of the hierarchy in the browser. -- contains many properties and methods that help you work with your browser - 'document' is the gateway to all the HTML elements - can read as well as manipulate the HTML element - can add/remove elements, set inline CSS styles, respond to mouse clicks/hovers etc The 'window' object is the top of the tree, and the 'document' object sits within this The document object represents our page
